# 🕒 TimerOnline - 실시간 공유 웹 타이머

**TimerOnline**은 누구나 링크 하나로 실시간 동기화되는 타이머와 스톱워치를 생성하고 공유할 수 있는 웹 어플리케이션입니다. 회의, 팀 프로젝트, 온라인 수업, 게임 등 여러 사람이 동시에 같은 시간을 확인해야 하는 모든 상황에 최적화되어 있습니다.

## ✨ 주요 기능

### 1. 실시간 동기화 (Real-time Sync)
- **Firebase Realtime Database**를 활용하여 모든 참여자가 소수점 단위까지 동일한 시간을 확인합니다.
- 방장(생성자)이 타이머를 일시정지하거나 재개하면 참여자들의 화면에도 즉시 반영됩니다.

### 2. 강력한 타이머 & 스톱워치 모드
- **카운트다운 (Timer)**
    - **시간 설정 모드**: 원하는 시간(시:분:초)을 직접 입력하여 시작.
    - **종료 시점 설정 모드**: 특정 날짜와 시간을 목표로 설정 (예: 시험 종료 시간, 퇴근 시간).
- **스톱워치 (Stopwatch)**
    - **바로 시작**: 0부터 즉시 측정 시작.
    - **시작 시점 설정 모드**: 정확히 지정된 시간에 자동으로 측정을 시작하는 예약 기능.

### 3. 직관적이고 몰입감 있는 UI/UX
- **압도적인 가독성**: 시간이 화면 정중앙에 거대하게 표시되어 멀리서도 한눈에 들어옵니다.
- **반응형 디자인**: PC, 태블릿, 모바일 등 어떤 기기에서도 최적의 비율로 숫자가 조절됩니다.
- **다크/라이트 테마 최적화**: 깔끔하고 세련된 디자인으로 시각적 피로도를 낮췄습니다.

### 4. 간편한 공유 및 관리
- **링크 공유**: 클릭 한 번으로 고유 대기실 링크를 복사하여 전달할 수 있습니다.
- **최근 목록 관리**: 내가 생성하거나 방문했던 타이머 목록을 하단에서 바로 확인할 수 있습니다.

## 🛠 기술 스택

- **Frontend**: Vanilla JS, HTML5, CSS3 (Custom Properties & Flexbox)
- **Backend**: Firebase Realtime Database
- **Design**: Responsive Typography (Clamp), Glassmorphism UI patterns

## 🚀 시작하기

1. **타이머 생성**: 메인 화면에서 원하는 모드(타이머/스톱워치)와 시간을 설정하고 '타이머 생성하기'를 누릅니다.
2. **링크 공유**: 우측 상단의 '링크 공유' 버튼을 눌러 링크를 친구나 동료에게 보냅니다.
3. **실시간 제어**: 방장은 타이머를 제어하고, 참여자는 실시간으로 업데이트되는 시간을 확인합니다.
